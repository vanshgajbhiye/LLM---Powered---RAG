# 🚀 RAG Chatbot using Mistral AI

An intelligent AI-powered chatbot built using *LangChain, **ChromaDB, and **Mistral AI* that can understand PDF documents and answer questions using Retrieval-Augmented Generation (RAG).

---

## ✨ Key Highlights

🔹 Upload and process PDF documents  
🔹 Generate semantic embeddings using Mistral AI  
🔹 Store embeddings in Chroma Vector Database  
🔹 Perform intelligent document retrieval using MMR  
🔹 Context-aware AI question answering  
🔹 Fast and efficient semantic search system  

---

## 🧠 Tech Stack

- Python
- LangChain
- ChromaDB
- Mistral AI
- PyPDFLoader
- RecursiveCharacterTextSplitter
